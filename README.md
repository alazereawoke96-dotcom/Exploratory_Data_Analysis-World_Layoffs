🌍 World Layoffs Exploratory Data Analysis

📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on a global layoffs dataset to identify patterns and trends in workforce reductions across companies, industries, and countries.
The objective is to clean, analyze, and visualize the dataset to better understand how layoffs are distributed across different sectors and time periods.
This project demonstrates practical data analytics skills including data cleaning, SQL analysis, and insight extraction from real-world datasets.

📂 Dataset
The dataset contains information about layoffs from companies around the world, including:
Company name
Industry
Country
Total layoffs
Percentage of layoffs
Date of layoffs
Funding stage

🔎 Key Analysis Performed
Companies with the highest number of layoffs
Layoffs by industry
Layoffs by country
Monthly and yearly layoff trends
